TA-Pagination

A light-weight, responsive and mobile first pagination for content, based on Alpine JS and Tailwind CSS. 100% customizable and endless animation options. If you already use Alpine JS and Tailwind CSS in your project you might consider using this pagination for articles, lists, products, news - all content that is formated as item list.

Pagination for articles, lists, products

In this example I used CSS grid to format all article examples. You can use different animations styles - check it out.
